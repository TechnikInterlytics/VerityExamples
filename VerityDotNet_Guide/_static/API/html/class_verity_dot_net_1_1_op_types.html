Detailed Description

Types for transform operation objects.Each operation category(OpCat) has one or more functions each of which is defined by an OpFunc object. Method SetUpCats is a convenience method to make list of OpCat objects, each of which defines a category of transform operations(Ops), and contains a child list of OpFunc objects.
