Processes a transform and uses LookUpDicts as needed. For lookups and by reference functions, there must also be passed in correlated lists of field objects, a dict of field titles (lower case) to index in list, each field's data type, and each field's current value in the source record. Lookups also require pre-processing the LookUp Dictionaries and passing into this function that list as well as the hash dictionary of the titles. They are ignored if not provided.

fieldValuesoptional list of values from parsed record per field
Returns
string of result value or starts with notok: if error
